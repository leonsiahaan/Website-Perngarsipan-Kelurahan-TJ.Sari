# Projek PKL : Website-Perngarsipan-Kelurahan-Tanjung-Sari
Aplikasi website ini merupakan projek PKL di kelurahan tanjung sari, Medan selayang, Medan. Sistem yang dibangun dapat dilakukan secara online tanpa perlu melibatkan pengunjung datang langsung ke Kantor Lurah. Terdapat admin yang ditugaskan sebagai Staff untuk melakukan acc kelengkapan data berkas pengunjung, kemudian Lurah bertugas sebagai Penandatangan SK.
Program dirancang menggunakan bahasa PHP, Javascript dan CSS.
